# LoopArchitect - Build Plan

Goal: Take a 4–8 bar loop upload and turn it into a full 3-minute instrumental (trap / dark ATL vibe).

## Phase A: Ingest + Analyze
- Upload audio loop (wav/mp3)
- Save file + metadata to DB
- Extract audio features:
- BPM
- Key
- duration
- loudness
- basic spectral stats

## Phase B: Arrange
- Create a structure template (Intro / Hook / Verse / Hook / Bridge / Outro)
- Use loop slicing + duplication + variation (filters, drops, stutters)
- Add drum patterns + 808 + hats + perc (MIDI generation)

## Phase C: Render
- Export stems + full mixdown
- Return download links

## Phase D: Frontend (later)
- Upload + preview + download stems
